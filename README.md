# Formation git

Résumé des commandes vues pendant la formation

## Commandes de bases

- git --version
- git config
- git help
- git init
- git status
- git add
- git commit
- git log
- git reflog

## Commandes du stash

- git stash save -u "WIP"
- git stash pop
- git stash drop
- git stash clear

## Commandes liées aux remotes

- git remote
- git push
- git pull
- git fetch

## Commandes liées aux branches

- git branch
- git switch
- git checkout
